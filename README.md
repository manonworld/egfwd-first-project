# EGFWD Reads Project

The EGFWD first project (Reads)


### Prerequisites

    1. Docker CE
    2. Docker Compose
    3. Recommended to install it on a Linux machine

    OR

    1. nodejs
    2. npm or yarn


### Installation

``` make install ```

OR

``` cd reads && npm install ```


### Run

``` make run ```

OR

``` cd reads && npm start ```


### Clean (only if you use docker)

``` make clean ```
