# EGFWD Reads Project

The EGFWD first project (Reads)


### Prerequisites

    1. nodejs
    2. npm or yarn


### Installation

For dockerized environment

Please run:

``` make install ```

For local environment

Please run:

``` npm install ``` or ``` yarn install ```

From within reads/ directory.


### Run

For dockerized environment

Please run:

``` make run ```

For local environment

Please run:

``` npm start ``` or ``` yarn start ```

From within reads/ directory.


### Clean

For dockerized environment only

Please run:

``` make clean ```
